# KnittingAssistant
A useful aid for knitters to use while creating projects - featuring a row counter, stitch counter, duration timer, stretch and water break reminders, and more ideas to come!

Something I like to do in my downtime is knit various projects. As of late, I made a pair of mittens and a sophie hood for my daughter. Currently, I use a pencil and paper to keep track of my rows. I feel this process could be made more efficient by creating a program that I can use on my phone to keep track of the the number of rows I have worked on, number of stitches for my current row, when I should get up and stretch / take a water break, etc.,

In summary, here are my ideas for this project to implement:
1. Row Counter - A simple button the user can press that increases the amount of rows by +1.
2. Stitch Tracker - Based on pattern or user input - keeps track of number of current stitches for the row being worked
3. PDF Scanner for knitting patterns - to keep track of the current row I am working on for my project and the approx. stitches I should have based on # of rows and the pattern itself
4. Duration Timer - To see how long it takes to complete a project and to find your average time worked per row for the project
5. Break Reminders - To promote healthy knitting! Make sure to get up and stretch your muscles and get some water!

I think I will start simple with this - a basic HTML page that has all the functionalities in place. I would then like to take that functionality and make it an app for tablets/ipads, phones, and some use on apple watches (row counter button and break reminders). This would also include adding some styling and animations to the app to make it "pretty" AND functional! 

If I can succeed in all this - I would also like to add customization/personalization for users to change the styling of their knitting pocket buddy! This means changing text styles, color schemes, sounds effects, etc.,
Lastly, included during the breaks, I think a hand stretching exercise with pictures for demonstration would be a nice touch!

Stay tuned for more!
